Readme.md
===============

This repository is to show a lightweight MVC application that allows the user to invoke the CRUD commands on a Person

## Requirements
* Java 8
* Maven 3.3
* Jetty 9
* SQLite 3.8

## Includes
* Spring WebMvc 4.2.1
* ORMLite 4.48
* Angular 1.4.5
* ngResource 1.4.5
* UI Router 0.2.15
* Bootstrap 3.3.5

## Usage
* Run with `mvn jetty:run-war`
